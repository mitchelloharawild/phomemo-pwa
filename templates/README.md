# Example SVG Templates

This folder contains example SVG templates demonstrating the special field types supported by the Phomemo Printer PWA.

## Available Examples

### 1. [`template-with-date.svg`](template-with-date.svg)
**Event Badge with Date Field**

Features:
- Text fields for attendee name and company
- Date field with custom format (`MMMM D, YYYY`)
- Clean, professional event badge design

Use cases:
- Event badges
- Conference passes
- Meeting name tags

### 2. [`template-with-qr.svg`](template-with-qr.svg)
**Product Label with QR Code**

Features:
- Product name and SKU text fields
- QR code field for product URL
- Multi-line description field
- Price field

Use cases:
- Product labels
- Inventory tags
- Retail price tags
- Item tracking labels

### 3. [`template-mixed-fields.svg`](template-mixed-fields.svg)
**Comprehensive Badge with All Field Types**

Features:
- Image upload field (for photos)
- Multiple text fields (name, title, company, etc.)
- Date field (conference date)
- QR code field (check-in code)
- All field types demonstrated in one template

Use cases:
- Conference badges with photos
- Employee ID cards
- VIP passes
- Multi-function access cards

### 4. [`template-optional-fields.svg`](template-optional-fields.svg)
**Label with Optional Fields**

Features:
- Optional date field in top-right corner
- Optional reference number in top-left corner
- Optional QR code in bottom-right
- Optional notes field in bottom-left
- Required name and description fields
- Demonstrates show/hide toggle functionality

Use cases:
- Flexible product labels
- Shipping labels (with/without date)
- General purpose labels
- Templates where not all fields are always needed
